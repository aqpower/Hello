### Problem A

A题是题意是给出三个数a b c，你可以选择其中两个数减一，另一个数加二，这样的操作你可以执行任意次，问a^2 + b^2 + c^2 最大的值为多少？

0 0 max  
1 0 max  
很显然，就这两种情况，我和队友也很快意识到就只有这两种情况

```C++
void solve()
{
    cin >> a[0] >> a[1] >> a[2];
    long long sum = a[0] + a[1] + a[2];
    sort(a,a + 3);
    if((a[2] - a[1]) % 3 == 0 || ( a[2] - a[0]) % 3 == 0 ||(a[1] - a[0]) % 3 == 0)
        cout << sum * sum << endl;
    else 
        cout << 1 + (sum - 1) * (sum - 1) << endl;
}
```
